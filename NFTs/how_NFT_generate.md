# NFT 制作和运营
在了解过 NFT 的本质和外在表现形式后，那站在项目方视角，NFT 是如何被制作出来的呢，它们是怎么运作的，我们这里做一个大致介绍。

## 参与者
先介绍 NFT 项目的几个重要参与角色。
- 创作者，是指为 NFT 具像化而创造的实物或数字物品，比如艺术作品、书籍、音乐等。
- 开发者：这里泛指为 NFT 的运营提供互联网技术的群体，包括技术、产品、设计等。
- 运营：这里指 NFT 项目方的运营人员。
